Array as DataStructure
    Continuous block of memory
    we can strore homogeneous of elements only
    We can store more than one element in an array
    Each and every memory block contains index value

Array Operation:
    Accessing -O(1)
    Insertion -O(1)/O(n)
    Deletion -O(n)/O(1)
    Traversing -O(n)
    Searching 
    Updation -O(1)

Benefit:
    Random access: O(1)
    Cache friendly 
    Ease of sorting
    Implements Other datastructure

Limitation:
    Fixed size
    O(n)-Insert, delete(worst case scenerio)
    Insufficient for frequent modification

UseCases:
    Implemented by other data structure
    Caching
    Visted 
    Lot of popular coding pattern
